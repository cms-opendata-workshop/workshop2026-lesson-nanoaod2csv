# Convert NANOAOD to CSV

## What is this?
In this notebook you will learn how to fetch data from the CERN Open Data Portal, read it in, make some simple selections, and export some information to a csv file.

## Local installation and usage
Install
```
git clone https://github.com/cms-opendata-workshop/workshop2026-lesson-nanoaod2csv.git
cd workshop2026-lesson-nanoaod
```
Use
```
virtualenv cms
source cms/bin/activate
jupyter lab
```
